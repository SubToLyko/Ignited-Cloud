# 🔥 Ignited Cloud

**Free Minecraft Server Hosting — No Limits, Just Play.**

Ignited Cloud is a free Minecraft hosting platform that lets players create and manage servers in seconds. Built on a custom Pterodactyl panel with Discord bot integration, we make server hosting fast, simple, and actually free.

---

## 🎮 What We Offer

- **Instant Server Setup** — Create a Minecraft server with one command
- **Discord Bot Control** — Start, stop, and manage your server right from Discord
- **Custom Panel** — Branded control panel (powered by Pterodactyl)
- **24/7 Uptime** — Servers stay online even when you're away
- **No Hidden Fees** — Actually free. No credit card. No trials. No BS.

---

## 🛠️ Tech Stack

| Layer | Technology |
|-------|------------|
| Game Panel | Pterodactyl |
| Discord Bot | Node.js (hosted on Railway) |
| Backend API | Express.js |
| Frontend | Custom branded panel |
| Server Nodes | Managed via Pterodactyl Wings (Docker) |
